# Miratuku × esse-sense Grant Schedule 2026

**ミラツク／esse-sense 補助金・助成金 提出スケジュール（2026年5-9月）**

公開ページ: https://yuyanishimura0312.github.io/miratuku-grant-schedule-2026/

## 機能

- **提出スケジュール表示** — 67件以上の助成金を時系列で整理
- **行編集** — クリックで内容を更新（ブラウザに自動保存）
- **ドラフトリンク** — 各助成金に提出書類ドラフトを紐付け
- **ダウンロード** — JSON、CSV、ドラフト一括ZIP
- **雛形アップロード自動書込** — `.docx`／`.txt`／`.md` の雛形にプレースホルダーを仕込めば、選択した助成金の情報を自動的に流し込んで生成
- **ダーク／ライトモード切替**
- **検索フィルタ**

## プレースホルダー仕様

雛形ファイル内に以下の形式でプレースホルダーを記入:

```
{{法人名}} {{代表者}} {{設立年}} {{所在地}} {{営業収益}} {{専従スタッフ数}}
{{助成金名}} {{締切}} {{上限額}} {{主体}} {{テーマ}} {{該当強み}}
{{研究テーマ}} {{事業概要}} {{助成希望額}} {{研究期間}}
```

`.docx` の場合は `{法人名}` のような単一波括弧形式も対応（docxtemplaterのデフォルト）。
