+# 観光庁 インバウンド受入環境整備高度化 申請書ドラフト（ミラツク向け）
+**締切**: 通年 | **上限**: 1,000万円(1/2)
+
+## 1. 申請者情報（300字・数値組込）
+特定非営利活動法人ミラツクは、2010年設立、第14期を迎える京都府・東京都を拠点とした非営利法人である。専従25名の体制により、企業協働20社、コミュニティ参加者350名と連携し、社会課題と事業創出を接続する実装型の活動を展開してきた。2024年度営業収益は87.4百万円であり、安定した事業運営基盤を有する。学術知識30,288概念を含む26のSQLite知識データベース、ならびに「事業のかたち」「暮らしのかたち」「変化のかたち」各100話、合計300話の知識資産を活用し、地域・企業・行政・研究者の協働を促進している。
+
+## 2. 助成金概要・応募要件（200字・地の文）
+本事業は、訪日外国人旅行者の周遊促進、消費拡大、地方誘客を目的に、観光地におけるまち歩きや広域周遊の受入環境整備を支援する制度である。高度化計画を策定する自治体、DMO、民間事業者等と連携し、計画に位置付けられた事業実施者として、観光DX、情報提供、多言語対応、回遊データ活用、滞在価値向上に資する取組を申請する。公募は予算状況により終了するため、最新回の締切確認を前提とする。
+
+## 3. 申請テーマ案（A推奨/B代替・各150字）
+**A推奨: 広域観光DXによる訪日客回遊・体験価値高度化事業**  
+京都・東京を起点に、地域観光資源、交通結節点、文化体験、滞在消費を結ぶデータ基盤を構築し、多言語コンテンツ、混雑・関心分析、モデルルート提示を一体化する。ミラツクの知識DBと実装力を活かし、訪日客の回遊と地域事業者の収益機会を高める。
+
+**B代替: 学際知識DBを活用した地域ストーリー型観光案内整備事業**  
+地域の歴史、暮らし、産業、文化を学際知識DBで再編集し、訪日客向けの多言語ストーリー、現地案内、デジタルマップとして実装する。観光地を単なる地点情報ではなく、地域の変化と生活文化を理解できる体験へ転換する。
+
+## 4. 事業/研究計画書本文（背景目的300-400字、方法300-400字、成果200-300字）
+訪日外国人旅行者の回復に伴い、主要観光地では混雑、情報偏在、短時間滞在、消費機会の限定が課題となっている。一方で、地域には生活文化、産業、研究知、コミュニティ活動など、既存の観光案内では十分に可視化されていない資源が多く存在する。本事業では、ミラツクが蓄積してきた26のSQLite知識データベース、学術知識30,288概念、300話の連載知見を活用し、訪日客の関心と地域資源を結び直す観光DX基盤を整備する。目的は、広域周遊を促す情報設計、地域事業者への送客、混雑分散、滞在満足度の向上を同時に実現し、観光地の受入環境を高度化することである。
+
+実施方法として、まず対象エリアの観光資源、交通動線、既存案内、地域事業者、訪日客ニーズを調査し、協働先自治体・DMO・民間事業者と高度化計画上の役割を整理する。次に、ミラツクのデータ活用力を用いて、地域資源を「文化」「暮らし」「産業」「学び」「体験」の観点から構造化し、多言語対応可能なコンテンツ原稿、モデルルート、デジタルマップ、QR導線を設計する。さらに、利用ログ、アンケート、事業者ヒアリングを組み合わせ、回遊傾向、関心領域、滞在消費の変化を分析する。実装段階では、現地掲示、Web案内、事業者紹介、体験予約導線を連動させ、観光DXを単なる情報発信ではなく、現地行動の変化に結び付ける。
+
+期待成果は、訪日客が地域内外を円滑に周遊し、文化理解と消費行動を深められる受入環境の整備である。定量面では、デジタル案内閲覧数、モデルルート利用数、協力事業者への送客数、満足度、再訪意向を指標化する。定性面では、地域資源の再編集、事業者間連携、行政・民間・NPOの協働体制を形成し、補助終了後も更新可能な観光DX運用モデルとして展開する。
+
+## 5. スケジュールと予算（年次別）
+初年度は、1-2か月目に協働先との合意形成、対象エリア選定、既存資料調査、訪日客ニーズ整理を行う。3-4か月目に地域資源データベース化、多言語コンテンツ設計、モデルルート案作成を進める。5-6か月目にデジタルマップ、QR導線、現地案内、事業者紹介ページを実装し、7-8か月目に試験運用、利用ログ分析、改善を実施する。9-12か月目に本運用、効果測定、成果報告、次年度展開計画を取りまとめる。
+
+予算は総事業費2,000万円、補助申請額1,000万円、自己負担1,000万円を想定する。主な内訳は、調査・設計費300万円、データ整備・分析費400万円、多言語コンテンツ制作費350万円、システム・デジタルマップ実装費500万円、現地案内・QR等整備費250万円、効果測定・報告費200万円とする。M+E体制により、ミラツクは企画、データ設計、協働形成、効果検証を担い、実装パートナーが開発・現地整備を担う。

# 観光庁 インバウンド受入環境整備高度化 申請書ドラフト（ミラツク向け）
**締切**: 通年 | **上限**: 1,000万円(1/2)

## 1. 申請者情報（300字・数値組込）
特定非営利活動法人ミラツクは、2010年設立、第14期を迎える京都府・東京都を拠点とした非営利法人である。専従25名の体制により、企業協働20社、コミュニティ参加者350名と連携し、社会課題と事業創出を接続する実装型の活動を展開してきた。2024年度営業収益は87.4百万円であり、安定した事業運営基盤を有する。学術知識30,288概念を含む26のSQLite知識データベース、ならびに「事業のかたち」「暮らしのかたち」「変化のかたち」各100話、合計300話の知識資産を活用し、地域・企業・行政・研究者の協働を促進している。

## 2. 助成金概要・応募要件（200字・地の文）
本事業は、訪日外国人旅行者の周遊促進、消費拡大、地方誘客を目的に、観光地におけるまち歩きや広域周遊の受入環境整備を支援する制度である。高度化計画を策定する自治体、DMO、民間事業者等と連携し、計画に位置付けられた事業実施者として、観光DX、情報提供、多言語対応、回遊データ活用、滞在価値向上に資する取組を申請する。公募は予算状況により終了するため、最新回の締切確認を前提とする。

## 3. 申請テーマ案（A推奨/B代替・各150字）
**A推奨: 広域観光DXによる訪日客回遊・体験価値高度化事業**  
京都・東京を起点に、地域観光資源、交通結節点、文化体験、滞在消費を結ぶデータ基盤を構築し、多言語コンテンツ、混雑・関心分析、モデルルート提示を一体化する。ミラツクの知識DBと実装力を活かし、訪日客の回遊と地域事業者の収益機会を高める。

**B代替: 学際知識DBを活用した地域ストーリー型観光案内整備事業**  
地域の歴史、暮らし、産業、文化を学際知識DBで再編集し、訪日客向けの多言語ストーリー、現地案内、デジタルマップとして実装する。観光地を単なる地点情報ではなく、地域の変化と生活文化を理解できる体験へ転換する。

## 4. 事業/研究計画書本文（背景目的300-400字、方法300-400字、成果200-300字）
訪日外国人旅行者の回復に伴い、主要観光地では混雑、情報偏在、短時間滞在、消費機会の限定が課題となっている。一方で、地域には生活文化、産業、研究知、コミュニティ活動など、既存の観光案内では十分に可視化されていない資源が多く存在する。本事業では、ミラツクが蓄積してきた26のSQLite知識データベース、学術知識30,288概念、300話の連載知見を活用し、訪日客の関心と地域資源を結び直す観光DX基盤を整備する。目的は、広域周遊を促す情報設計、地域事業者への送客、混雑分散、滞在満足度の向上を同時に実現し、観光地の受入環境を高度化することである。

実施方法として、まず対象エリアの観光資源、交通動線、既存案内、地域事業者、訪日客ニーズを調査し、協働先自治体・DMO・民間事業者と高度化計画上の役割を整理する。次に、ミラツクのデータ活用力を用いて、地域資源を「文化」「暮らし」「産業」「学び」「体験」の観点から構造化し、多言語対応可能なコンテンツ原稿、モデルルート、デジタルマップ、QR導線を設計する。さらに、利用ログ、アンケート、事業者ヒアリングを組み合わせ、回遊傾向、関心領域、滞在消費の変化を分析する。実装段階では、現地掲示、Web案内、事業者紹介、体験予約導線を連動させ、観光DXを単なる情報発信ではなく、現地行動の変化に結び付ける。

期待成果は、訪日客が地域内外を円滑に周遊し、文化理解と消費行動を深められる受入環境の整備である。定量面では、デジタル案内閲覧数、モデルルート利用数、協力事業者への送客数、満足度、再訪意向を指標化する。定性面では、地域資源の再編集、事業者間連携、行政・民間・NPOの協働体制を形成し、補助終了後も更新可能な観光DX運用モデルとして展開する。

## 5. スケジュールと予算（年次別）
初年度は、1-2か月目に協働先との合意形成、対象エリア選定、既存資料調査、訪日客ニーズ整理を行う。3-4か月目に地域資源データベース化、多言語コンテンツ設計、モデルルート案作成を進める。5-6か月目にデジタルマップ、QR導線、現地案内、事業者紹介ページを実装し、7-8か月目に試験運用、利用ログ分析、改善を実施する。9-12か月目に本運用、効果測定、成果報告、次年度展開計画を取りまとめる。

予算は総事業費2,000万円、補助申請額1,000万円、自己負担1,000万円を想定する。主な内訳は、調査・設計費300万円、データ整備・分析費400万円、多言語コンテンツ制作費350万円、システム・デジタルマップ実装費500万円、現地案内・QR等整備費250万円、効果測定・報告費200万円とする。M+E体制により、ミラツクは企画、データ設計、協働形成、効果検証を担い、実装パートナーが開発・現地整備を担う。
