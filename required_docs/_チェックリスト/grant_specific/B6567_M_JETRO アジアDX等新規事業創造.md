+# JETRO アジアDX等新規事業創造 申請書ドラフト（ミラツク向け）
+**締切**: 通年 | **上限**: 要公式確認
+
+## 1. 申請者情報（300字・数値組込）
+特定非営利活動法人ミラツクは、2010年設立、第14期を迎える京都府・東京都拠点の事業開発・知識創造組織である。専従25名体制で、2024年度営業収益は87.4M円。企業協働20社、コミュニティ参加者350名を基盤に、企業、研究者、行政、市民を接続する共創型事業を推進してきた。知識基盤として26のSQLite知識データベースを保有し、学術知識30,288概念を含む構造化データを活用する。さらに「事業のかたち」「暮らしのかたち」「変化のかたち」各100話、計300話の連載を通じ、知の社会接続と実装知の可視化を進めている。
+
+## 2. 助成金概要・応募要件（200字・地の文）
+本制度は、日本法人がアジア地域の企業・団体等と連携し、デジタル技術を用いて現地社会課題の解決と新規事業創造を進める実証を支援する趣旨の国系補助事業である。対象分野はDXを核とするヘルスケア、教育、金融、物流、脱炭素等の社会課題領域が想定される。応募時には日本側申請主体、海外共創先、実証内容、事業化見込み、経費計画、実施体制を具体化する必要がある。最新の公募状況、締切、上限額、対象国はJETRO公式情報で確認する。
+
+## 3. 申請テーマ案（A推奨/B代替・各150字）
+**A推奨: アジア社会課題ナレッジDX共創基盤の実証**  
+ミラツクの学際DBと大規模解析技術を用い、アジア現地パートナーの社会課題、研究知、生活者知、事業仮説を統合する共創基盤を構築する。教育・地域産業・脱炭素等の課題を対象に、現地で使える事業化候補を抽出し、PoCから継続事業化までを設計する。
+
+**B代替: 現地NPO・企業向けDX型事業開発プログラム**  
+アジアの現地NPO、企業、行政関係者を対象に、課題探索、知識DB活用、事業仮説生成、実証計画作成を一体化したDX型事業開発プログラムを実施する。日本側企業との協働機会を組み込み、社会課題解決と市場形成の両立を図る。
+
+## 4. 事業/研究計画書本文（背景目的300-400字、方法300-400字、成果200-300字）
+**背景目的**  
+アジア各国では、都市化、教育格差、地域産業の転換、気候変動対応などの課題が複合化している。一方、現地の課題情報、研究知、企業技術、生活者の経験は分断され、実証可能な事業仮説へ変換されにくい。ミラツクは、26のSQLite知識データベース、学術知識30,288概念、300話の社会実装連載、企業協働20社のネットワークを有し、知識を社会実装へ接続する方法論を蓄積してきた。本事業では、アジア現地パートナーと連携し、社会課題を構造化し、DXを用いて事業仮説を生成・検証する共創基盤を実証する。目的は、単なる調査ではなく、現地で継続可能な新規事業の候補を形成し、日本側企業・研究者・実装主体が参加できる国際DX共創モデルを確立することである。
+
+**方法**  
+第1段階では、対象国・対象課題を1から2領域に絞り、現地企業、NPO、行政関係者、研究者へのヒアリングを実施する。収集した課題文脈をミラツクの学際DBと照合し、関連概念、先行事例、技術シーズ、事業モデルを構造化する。第2段階では、データ解析とワークショップを組み合わせ、現地パートナーと共同で複数の事業仮説を設計する。仮説は、課題適合性、収益可能性、現地運用性、日本側協働可能性の観点で評価する。第3段階では、最有力案について小規模PoCを実施し、ユーザー反応、運用負荷、データ取得可能性、事業継続条件を検証する。ミラツクは全体設計、知識DB分析、共創プロセス設計、成果整理を担い、海外招聘・現地対話も活用して実装体制を強化する。
+
+**成果**  
+成果として、対象国・対象領域における社会課題マップ、事業仮説リスト、PoC実施報告、事業化ロードマップ、共創パートナー候補リストを作成する。少なくとも1件は、現地パートナーと日本側協働者が継続検討できる事業案として具体化する。加えて、ミラツクの知識DBを国際DX事業創造に適用する手順を標準化し、他国・他領域へ展開可能な実装モデルとして整理する。
+
+## 5. スケジュールと予算（年次別）
+**1年目**は、公式公募要件確認、対象国・現地パートナー確定、課題ヒアリング、DB分析、共創ワークショップ、PoC設計、実証、成果報告までを実施する。予算は、人件費、外部専門家謝金、現地調査旅費、通訳・翻訳費、データ整理・分析費、ワークショップ運営費、PoC外注費、成果物作成費を中心に組む。補助上限と補助率は公式確認後に調整する。
+
+**2年目以降**は、補助対象外または次段階資金を想定し、PoC結果をもとに共同事業化、追加国展開、企業協働20社ネットワークへの接続、国際共同研究・事業開発案件への発展を進める。自走化に向け、現地運営費、日本側企業負担、共同研究費、サービス利用料の組み合わせを検討する。

+アジア各国では、都市化、教育格差、地域産業の転換、気候変動対応などの課題が複合化している。一方、現地の課題情報、研究知、企業技術、生活者の経験は分断され、実証可能な事業仮説へ変換されにくい。ミラツクは、26のSQLite知識データベース、学術知識30,288概念、300話の社会実装連載、企業協働20社のネットワークを有し、知識を社会実装へ接続する方法論を蓄積してきた。本事業では、アジア現地パートナーと連携し、社会課題を構造化し、DXを用いて事業仮説を生成・検証する共創基盤を実証する。目的は、単なる調査ではなく、現地で継続可能な新規事業の候補を形成し、日本側企業・研究者・実装主体が参加できる国際DX共創モデルを確立することである。
+
+**方法**  
+第1段階では、対象国・対象課題を1から2領域に絞り、現地企業、NPO、行政関係者、研究者へのヒアリングを実施する。収集した課題文脈をミラツクの学際DBと照合し、関連概念、先行事例、技術シーズ、事業モデルを構造化する。第2段階では、データ解析とワークショップを組み合わせ、現地パートナーと共同で複数の事業仮説を設計する。仮説は、課題適合性、収益可能性、現地運用性、日本側協働可能性の観点で評価する。第3段階では、最有力案について小規模PoCを実施し、ユーザー反応、運用負荷、データ取得可能性、事業継続条件を検証する。ミラツクは全体設計、知識DB分析、共創プロセス設計、成果整理を担い、海外招聘・現地対話も活用して実装体制を強化する。
+
+**成果**  
+成果として、対象国・対象領域における社会課題マップ、事業仮説リスト、PoC実施報告、事業化ロードマップ、共創パートナー候補リストを作成する。少なくとも1件は、現地パートナーと日本側協働者が継続検討できる事業案として具体化する。加えて、ミラツクの知識DBを国際DX事業創造に適用する手順を標準化し、他国・他領域へ展開可能な実装モデルとして整理する。
+
