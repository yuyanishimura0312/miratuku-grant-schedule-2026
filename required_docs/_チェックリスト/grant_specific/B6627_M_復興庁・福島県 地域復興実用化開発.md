+# 復興庁・福島県 地域復興実用化開発 申請書ドラフト（ミラツク向け）
+**締切**: 通年 | **上限**: 施策毎

+## 1. 申請者情報（300字・数値組込）
+特定非営利活動法人ミラツクは、2010年設立、現在第14期を迎える京都府・東京都を拠点とする実践型の知識創造組織である。専従25名の体制で、2024年度営業収益は87.4百万円。企業協働は20社、コミュニティ参加者は350名に及び、研究者、企業、行政、地域実践者を結ぶ事業を継続してきた。学術知識30,288概念を含む26のSQLite知識データベースを整備し、「事業のかたち」「暮らしのかたち」「変化のかたち」各100話、合計300話の連載を通じて、知の社会接続と実装知の蓄積を進めている。

+## 2. 助成金概要・応募要件（200字・地の文）
+本事業は、福島イノベーション・コースト構想の重点分野に関し、福島県浜通り地域等における実用化開発、実証、事業化に向けた取組を支援する制度である。対象は地元企業等、または地元企業等と連携する法人等であり、廃炉、ロボット・ドローン、エネルギー・環境・リサイクル、農林水産業、医療関連、航空宇宙等の分野で、地域産業の回復と新産業創出に資する計画が求められる。
+
+## 3. 申請テーマ案（A推奨/B代替・各150字）
+A推奨：福島浜通り地域の復興事業・生活再建・産業創出に関する実践知を、ミラツクの学際データベースと企業協働ネットワークで構造化し、地元企業の新規事業開発、実証設計、事業化判断を支援する「復興実装ナレッジ基盤」を開発する。
+
+B代替：農林水産業、環境・リサイクル、医療関連等の重点分野を横断し、地域事業者・研究者・企業が保有する課題と技術を可視化する共創型マッチング基盤を構築する。浜通り地域での実証案件形成を促進し、復興事業の継続的創出を図る。
+
+## 4. 事業/研究計画書本文（背景目的300-400字、方法300-400字、成果200-300字）
+背景目的：福島県浜通り地域等では、震災・原子力災害からの産業回復に加え、人口減少、担い手不足、地域内外の知識接続不足という複合課題に直面している。一方で、地域には復興過程で蓄積された実践知、研究機関や企業の技術、生活者の経験が存在するが、それらは個別事業や関係者の経験にとどまり、次の実用化開発へ再利用しにくい。ミラツクは26のSQLite知識データベース、学術知識30,288概念、20社の企業協働、350名のコミュニティを活用し、地域課題と技術シーズを結び直す。本事業では、復興に資する実装知を構造化し、地元企業等が実証から事業化へ進むための判断材料と協働設計手法を開発する。
+
+方法：第一に、浜通り地域等の地元企業、自治体、研究者、生活者へのヒアリングを行い、重点分野別に課題、技術、事業化条件、規制・市場要因を整理する。第二に、ミラツクが保有する学際データベースと300話の連載知見を用いて、復興事業に関係する概念、事例、協働主体、実証プロセスをタグ化し、検索・比較可能な知識基盤を構築する。第三に、企業協働20社のネットワークを活用し、地元企業等との共創ワークショップを実施する。候補テーマごとに、実証仮説、必要技術、利用者価値、収益化条件、地域波及効果を整理し、プロトタイプ開発または実証計画書に落とし込む。第四に、実証結果をデータ化し、継続的な改善と他地域展開に耐えるモデルとして整備する。
+
+成果：成果物は、復興実装ナレッジ基盤、重点分野別の実証テーマ候補、地元企業等との協働プロトタイプ、事業化判断レポートである。短期的には浜通り地域における実用化開発案件の形成数を増やし、中期的には地域企業が外部技術・知識を活用して新規事業を立ち上げる仕組みを確立する。ミラツクの強みである学際DB、セクター協働、実装力を通じ、復興を単発支援ではなく継続的な産業創出へ接続する。

+1年目は、対象地域・重点分野の調査、地元企業等との連携確認、ヒアリング、既存データベースの再編、共創ワークショップを実施する。主な予算は人件費、調査旅費、外部専門家謝金、データ整理費、ワークショップ運営費とする。

本事業は、福島イノベーション・コースト構想の重点分野に関し、福島県浜通り地域等における実用化開発、実証、事業化に向けた取組を支援する制度である。対象は地元企業等、または地元企業等と連携する法人等であり、廃炉、ロボット・ドローン、エネルギー・環境・リサイクル、農林水産業、医療関連、航空宇宙等の分野で、地域産業の回復と新産業創出に資する計画が求められる。

## 3. 申請テーマ案（A推奨/B代替・各150字）
A推奨：福島浜通り地域の復興事業・生活再建・産業創出に関する実践知を、ミラツクの学際データベースと企業協働ネットワークで構造化し、地元企業の新規事業開発、実証設計、事業化判断を支援する「復興実装ナレッジ基盤」を開発する。

B代替：農林水産業、環境・リサイクル、医療関連等の重点分野を横断し、地域事業者・研究者・企業が保有する課題と技術を可視化する共創型マッチング基盤を構築する。浜通り地域での実証案件形成を促進し、復興事業の継続的創出を図る。

## 4. 事業/研究計画書本文（背景目的300-400字、方法300-400字、成果200-300字）
背景目的：福島県浜通り地域等では、震災・原子力災害からの産業回復に加え、人口減少、担い手不足、地域内外の知識接続不足という複合課題に直面している。一方で、地域には復興過程で蓄積された実践知、研究機関や企業の技術、生活者の経験が存在するが、それらは個別事業や関係者の経験にとどまり、次の実用化開発へ再利用しにくい。ミラツクは26のSQLite知識データベース、学術知識30,288概念、20社の企業協働、350名のコミュニティを活用し、地域課題と技術シーズを結び直す。本事業では、復興に資する実装知を構造化し、地元企業等が実証から事業化へ進むための判断材料と協働設計手法を開発する。

方法：第一に、浜通り地域等の地元企業、自治体、研究者、生活者へのヒアリングを行い、重点分野別に課題、技術、事業化条件、規制・市場要因を整理する。第二に、ミラツクが保有する学際データベースと300話の連載知見を用いて、復興事業に関係する概念、事例、協働主体、実証プロセスをタグ化し、検索・比較可能な知識基盤を構築する。第三に、企業協働20社のネットワークを活用し、地元企業等との共創ワークショップを実施する。候補テーマごとに、実証仮説、必要技術、利用者価値、収益化条件、地域波及効果を整理し、プロトタイプ開発または実証計画書に落とし込む。第四に、実証結果をデータ化し、継続的な改善と他地域展開に耐えるモデルとして整備する。

成果：成果物は、復興実装ナレッジ基盤、重点分野別の実証テーマ候補、地元企業等との協働プロトタイプ、事業化判断レポートである。短期的には浜通り地域における実用化開発案件の形成数を増やし、中期的には地域企業が外部技術・知識を活用して新規事業を立ち上げる仕組みを確立する。ミラツクの強みである学際DB、セクター協働、実装力を通じ、復興を単発支援ではなく継続的な産業創出へ接続する。

## 5. スケジュールと予算（年次別）
1年目は、対象地域・重点分野の調査、地元企業等との連携確認、ヒアリング、既存データベースの再編、共創ワークショップを実施する。主な予算は人件費、調査旅費、外部専門家謝金、データ整理費、ワークショップ運営費とする。

2年目は、知識基盤の試作、実証テーマの選定、プロトタイプ開発、地元企業等との実証運用、評価指標の設計を進める。主な予算は開発委託費、実証経費、データ分析費、地域調整費、成果広報費とする。

3年目は、実証結果の改善、事業化計画の作成、地元企業への移管・共同運営体制の設計、横展開可能な手順書の整備を行う。主な予算は事業化支援費、知財・契約整理費、成果報告書作成費、普及展開費とする。
