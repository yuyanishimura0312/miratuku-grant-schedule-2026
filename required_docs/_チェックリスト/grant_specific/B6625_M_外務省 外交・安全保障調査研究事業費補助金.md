+# 外務省 外交・安全保障調査研究事業費補助金 申請書ドラフト（ミラツク向け）
+**締切**: 通年 | **上限**: テーマ毎
+
+## 1. 申請者情報（300字・数値組込）
+特定非営利活動法人ミラツクは、2010年設立、第14期を迎える京都府・東京都拠点の非営利法人である。専従25名体制で、2024年度営業収益は87.4百万円。企業協働20社、コミュニティ参加者350名を基盤に、異分野の知見を社会課題の実装へ接続してきた。知識基盤として26のSQLite知識データベースを整備し、学術知識30,288概念等を蓄積している。また「事業のかたち」「暮らしのかたち」「変化のかたち」各100話、合計300話の連載を通じ、専門知を市民・企業・政策関係者へ翻訳し発信してきた。
+
+## 2. 助成金概要・応募要件（200字・地の文）
+本補助金は、外交・安全保障に関する我が国の調査研究機関の情報収集、分析、発信、政策提案能力を高め、日本の総合的外交力の強化に資することを目的とする。対象は、政策志向の基礎的調査研究、国内外への機動的発信、外国シンクタンク・有識者との連携、国際世論形成への貢献等である。申請に当たっては、最新公募テーマ、受付期間、代表法人要件、共同申請可否、補助対象経費を必ず確認する。
+
+## 3. 申請テーマ案（A推奨/B代替・各150字）
+**A推奨: データ駆動型ナレッジ外交による日本発の社会課題解決モデルの国際発信**  
+ミラツクの学際DBと大規模解析力を用い、気候、人口、ケア、地域産業等の社会課題に関する日本発の実装知を整理し、海外有識者との対話を通じて国際発信する。
+
+**B代替: 東アジア・グローバルサウス連携に向けた市民社会型外交知の構築**  
+企業、研究者、NPO、市民が協働する日本の実践知を比較分析し、国際セミナー、英語レポート、政策提言により、包摂的な外交・安全保障議論へ接続する。
+
+## 4. 事業/研究計画書本文（背景目的300-400字、方法300-400字、成果200-300字）
+国際秩序が流動化する中、外交・安全保障は政府間交渉だけでなく、技術、生活、地域、産業、市民社会の知を統合した総合的な国際発信を必要としている。一方、日本の現場には、防災、地域再生、ケア、環境、産業共創など、国際社会に共有し得る実装知が蓄積されているが、学術知、企業実践、市民の経験が分断され、外交的文脈で再編集される機会は限られている。本事業は、ミラツクが保有する26のSQLite知識データベースと30,288概念の学際知識、300話の社会実装連載、企業協働20社・参加者350名のネットワークを活用し、日本発の社会課題解決モデルを外交・安全保障領域の国際発信資源として体系化することを目的とする。
+
+方法として、第一に既存DBと連載記事を対象に、外交・安全保障に関係する社会課題、技術動向、制度設計、地域実装の概念群を抽出し、定量的な共起分析と専門家レビューにより論点地図を作成する。第二に、国内の企業、研究者、NPO、行政関係者へのヒアリングを行い、国際発信に耐える事例、成功条件、移転可能性を整理する。第三に、海外有識者・シンクタンク関係者を招聘またはオンライン招聘し、日本の実装知に対する国際的評価、比較視点、政策含意を検証する。第四に、英語ブリーフ、公開ウェビナー、政策提言書を作成し、外務省事業の趣旨に沿って、機動的かつタイムリーな国内外発信を行う。
+
+成果として、外交・安全保障上の国際発信に活用できる日本発社会課題解決モデルの論点地図、英語政策ブリーフ、国内外向け公開セミナー、海外有識者コメントを含む最終報告書を作成する。大規模解析と現場ネットワークを組み合わせることで、従来の政策研究に加え、市民社会・企業協働を含む全員参加型外交の具体的な知的基盤を提示する。

+## 5. スケジュールと予算（年次別）
+**1年目**: 公募テーマ精査、研究設計、DB抽出、国内ヒアリング、論点地図作成、海外有識者候補選定。予算は人件費、調査補助費、データ加工費、国内旅費、謝金を中心に配分する。  
+**2年目**: 海外有識者招聘、国際比較レビュー、英語ブリーフ作成、公開ウェビナー、政策提言骨子の作成。予算は国外・国内旅費、会議運営費、翻訳・校正費、成果広報費を中心とする。  
+**3年目**: 最終報告書、政策提言書、英語発信、公開シンポジウム、成果評価を実施する。予算は報告書作成費、HP掲載費、成果広報費、事業管理費に充当する。事業管理費は対象経費合計の10％以内で設計する。

ミラツクの学際DBと大規模解析力を用い、気候、人口、ケア、地域産業等の社会課題に関する日本発の実装知を整理し、海外有識者との対話を通じて国際発信する。

**B代替: 東アジア・グローバルサウス連携に向けた市民社会型外交知の構築**  
企業、研究者、NPO、市民が協働する日本の実践知を比較分析し、国際セミナー、英語レポート、政策提言により、包摂的な外交・安全保障議論へ接続する。

## 4. 事業/研究計画書本文（背景目的300-400字、方法300-400字、成果200-300字）
国際秩序が流動化する中、外交・安全保障は政府間交渉だけでなく、技術、生活、地域、産業、市民社会の知を統合した総合的な国際発信を必要としている。一方、日本の現場には、防災、地域再生、ケア、環境、産業共創など、国際社会に共有し得る実装知が蓄積されているが、学術知、企業実践、市民の経験が分断され、外交的文脈で再編集される機会は限られている。本事業は、ミラツクが保有する26のSQLite知識データベースと30,288概念の学際知識、300話の社会実装連載、企業協働20社・参加者350名のネットワークを活用し、日本発の社会課題解決モデルを外交・安全保障領域の国際発信資源として体系化することを目的とする。

方法として、第一に既存DBと連載記事を対象に、外交・安全保障に関係する社会課題、技術動向、制度設計、地域実装の概念群を抽出し、定量的な共起分析と専門家レビューにより論点地図を作成する。第二に、国内の企業、研究者、NPO、行政関係者へのヒアリングを行い、国際発信に耐える事例、成功条件、移転可能性を整理する。第三に、海外有識者・シンクタンク関係者を招聘またはオンライン招聘し、日本の実装知に対する国際的評価、比較視点、政策含意を検証する。第四に、英語ブリーフ、公開ウェビナー、政策提言書を作成し、外務省事業の趣旨に沿って、機動的かつタイムリーな国内外発信を行う。

成果として、外交・安全保障上の国際発信に活用できる日本発社会課題解決モデルの論点地図、英語政策ブリーフ、国内外向け公開セミナー、海外有識者コメントを含む最終報告書を作成する。大規模解析と現場ネットワークを組み合わせることで、従来の政策研究に加え、市民社会・企業協働を含む全員参加型外交の具体的な知的基盤を提示する。

## 5. スケジュールと予算（年次別）
**1年目**: 公募テーマ精査、研究設計、DB抽出、国内ヒアリング、論点地図作成、海外有識者候補選定。予算は人件費、調査補助費、データ加工費、国内旅費、謝金を中心に配分する。  
**2年目**: 海外有識者招聘、国際比較レビュー、英語ブリーフ作成、公開ウェビナー、政策提言骨子の作成。予算は国外・国内旅費、会議運営費、翻訳・校正費、成果広報費を中心とする。  
**3年目**: 最終報告書、政策提言書、英語発信、公開シンポジウム、成果評価を実施する。予算は報告書作成費、HP掲載費、成果広報費、事業管理費に充当する。事業管理費は対象経費合計の10％以内で設計する。
