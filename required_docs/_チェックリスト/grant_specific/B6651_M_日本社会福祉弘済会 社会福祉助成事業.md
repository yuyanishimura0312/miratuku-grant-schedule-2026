+# 日本社会福祉弘済会 社会福祉助成事業 申請書ドラフト（ミラツク向け）
+**締切**: 通年 | **上限**: 50万円

+## 1. 申請者情報（300字・数値組込）
+特定非営利活動法人ミラツクは、2010年設立、第14期を迎えるNPO法人であり、京都府および東京都を拠点に、社会課題の解決に向けた知識創造、対話、実装支援を行っている。専従職員25名を擁し、2024年度営業収益は87.4百万円である。主要事業では、企業協働20社、コミュニティ参加者350名との連携を通じ、福祉、教育、地域、産業を横断する実践知の蓄積と活用を進めてきた。知識基盤として26のSQLite知識データベースを整備し、学術知識30,288概念を含む多分野の知を社会実装に接続している。また、「事業のかたち」「暮らしのかたち」「変化のかたち」各100話、合計300話の連載を通じ、現場の実践を可視化し、学び合いの資源として社会に還元している。
+
+## 2. 助成金概要・応募要件（200字・地の文）
+日本社会福祉弘済会の社会福祉助成事業は、社会福祉の向上に資する研修、研究、実践活動等を支援する助成制度である。本申請では、NPOが実施主体となり、福祉分野に関わる支援者、地域団体、企業、行政等の学びを深める「B 福祉研修」に焦点を当てる。上限50万円の範囲で、研修設計、教材開発、講師謝金、記録作成、成果共有に活用し、ミラツクの強みである知の社会接続とデータ活用を福祉現場の実践力向上に結びつける。
+
+## 3. 申請テーマ案（A推奨/B代替・各150字）
+**A推奨: 福祉現場の実践知を可視化する越境型研修プログラム**  
+福祉現場の支援者、NPO、企業、行政関係者が参加し、支援の経験、困難、判断基準を対話とデータ整理により可視化する研修を実施する。ミラツクの知識DBと連載資産を活用し、現場で使える実践知として再編集する。
+
+**B代替: 多職種協働を促す福祉研修教材の開発と試行**  
+地域福祉に関わる多職種が共通言語を持つため、事例読解、対話、概念整理を組み合わせた研修教材を開発する。福祉、暮らし、事業の接点を扱い、参加者が自組織で展開できる小規模研修モデルとして試行する。
+
+## 4. 事業/研究計画書本文（背景目的300-400字、方法300-400字、成果200-300字）
+**背景目的**  
+高齢化、孤立、生活困窮、障害、ケア人材不足など、福祉課題は複合化し、単一の制度や専門職だけでは対応しきれない状況が広がっている。一方、福祉現場には日々の支援から生まれる豊かな実践知が蓄積されているが、多忙な業務の中で言語化、共有、継承されにくい。ミラツクはこれまで、学術知識30,288概念を含む26の知識DB、300話の連載、20社との企業協働、350名規模のコミュニティを通じ、異なる領域の知を社会実装へつなぐ活動を進めてきた。本事業では、福祉現場の実践知を可視化し、多職種が相互に学び合う研修を実施することで、支援者の判断力、協働力、実装力を高めることを目的とする。
+
+**方法**  
+本事業は、準備、研修実施、記録整理、成果共有の4段階で進める。準備段階では、福祉現場の支援者や関係団体へのヒアリングを行い、研修で扱う主要課題を抽出する。あわせて、ミラツクが有する学際DBと連載記事から、福祉、暮らし、地域、事業形成に関わる概念や事例を選定し、研修教材として再編集する。研修は全2回から3回程度とし、参加者が自らの実践を持ち寄り、対話、概念整理、ケース検討を行う形式とする。各回では、支援場面の判断、連携の難しさ、本人中心の支援、地域資源との接続を扱い、参加者が自組織に持ち帰れる行動案を作成する。終了後は記録を整理し、個人情報に配慮したうえで、研修モデル、教材、実践知の要点をまとめる。
+
+**成果**  
+成果として、福祉研修プログラム一式、研修教材、実践知整理シート、成果報告書を作成する。参加者は、現場経験を言語化し、他職種と共有する方法を学ぶことで、日常業務の振り返りと連携改善に活用できる。ミラツクにとっては、知識DBと社会福祉現場を接続する実践モデルが形成され、今後の地域福祉、企業協働、NPO支援への展開基盤となる。
