# 内閣府 新しい地方経済・生活環境創生交付金(第2世代) 申請書ドラフト（ミラツク向け）
**締切**: 通年 | **上限**: 数千万-数億円

## 1. 申請者情報（300字・数値組込）
特定非営利活動法人ミラツクは、2010年設立、第14期を迎える社会的事業創出支援組織であり、京都府・東京都を拠点に、専従25名体制で活動している。2024年度営業収益は87.4百万円で、企業協働20社、コミュニティ参加者350名を基盤に、自治体、企業、研究者、市民をつなぐ共創型プロジェクトを継続してきた。学術知識30,288概念を含む26のSQLite知識データベースと、「事業のかたち」「暮らしのかたち」「変化のかたち」各100話、合計300話の知見蓄積を有し、地域課題をデータと対話から構造化し、実装へ結びつけることを強みとする。

## 2. 助成金概要・応募要件（200字・地の文）
本交付金は、地方公共団体が主体となり、地域の稼ぐ力、生活環境、担い手形成を一体的に高める地方創生事業を支援する国の制度である。ミラツクは自治体を申請主体とし、民間連携、データ分析、住民・企業参画設計、実証運営を担う共同実施者として位置づける。通年での制度活用を見据え、地域再生計画や総合戦略との整合を明確にする。

## 3. 申請テーマ案（A推奨/B代替・各150字）
A推奨案は「地域産業と生活課題をつなぐ共創データ基盤による地方経済循環モデル構築」である。自治体の産業政策、移住定住、福祉、教育、観光等の課題を横断的に解析し、企業20社規模の協働ネットワークと地域人材を接続する。

B代替案は「知識データベースを活用した地域未来人材・共創拠点形成事業」である。26の知識DBと300話の事例知を活用し、自治体職員、地域企業、住民が課題を言語化し、実証テーマを継続的に生み出す仕組みを整備する。

## 4. 事業/研究計画書本文（背景目的300-400字、方法300-400字、成果200-300字）
地方部では、人口減少、担い手不足、地域産業の付加価値停滞、生活サービス維持の困難が同時に進行している。一方で、自治体の政策情報、企業の現場知、住民の生活実感、大学等の研究知は分断され、個別事業の積み上げにとどまりやすい。ミラツクは2010年以降、京都府・東京都を拠点に、企業20社、参加者350名の共創コミュニティを形成してきた。本事業では、自治体を申請主体とし、ミラツクの学際DB、セクター協働、データ活用力を組み合わせ、地域課題を可視化し、産業振興と生活環境改善を同時に進める地方創生モデルを構築する。

方法として、第一に、自治体の総合戦略、産業統計、住民アンケート、既存施策を収集し、ミラツクが保有する26のSQLite知識データベースと照合して課題構造を整理する。第二に、地域企業、金融機関、教育機関、住民団体、行政部局を対象にワークショップを実施し、実装可能なテーマを抽出する。第三に、抽出テーマを地域産品の高付加価値化、生活サービス再設計、関係人口創出、人材育成等の実証事業に落とし込み、KPI、役割分担、資金計画を設定する。第四に、300話の連載知見を参照しながら実証過程を記録・分析し、自治体が自走できる運営手順書とデータ更新ルールを整備する。

期待される成果は、自治体が政策分野を越えて地域課題を把握し、民間企業や住民と実装につなげる共創基盤を獲得することである。短期的には複数の実証テーマ、参加企業・団体、住民参画機会を創出し、中期的には地域内経済循環、生活環境改善、担い手育成に関するKPIを継続管理する。ミラツクのデータ活用と協働設計により、交付金終了後も活用可能な地方創生の実装モデルを残す。

1年目は、自治体との協定締結、既存計画との整合確認、基礎データ収集、関係者ヒアリング、課題構造分析、共創ワークショップを行う。予算は調査分析費、会議運営費、専門人材費、データ整備費を中心に、数千万円規模を想定する。
