# 総務省 ローカル5G関連事業（地域実装支援） 申請書ドラフト（ミラツク向け）
**締切**: 通年 | **上限**: 数千万円

## 1. 申請者情報（300字・数値組込）
特定非営利活動法人ミラツクは、2010年設立、第14期を迎える京都府・東京都拠点の社会実装支援組織である。専従25名体制で、2024年度営業収益は87.4M円。企業協働20社、コミュニティ参加者350名との共創基盤を持ち、26のSQLite知識データベースに学術知識30,288概念を蓄積している。連載「事業のかたち」「暮らしのかたち」「変化のかたち」各100話、合計300話を通じ、研究知と現場知を社会課題の実装に接続してきた。学際DB、データ活用、実装支援を核に、自治体・企業・研究者を結ぶ地域DXの設計と伴走を担う。

## 2. 助成金概要・応募要件（200字・地の文）
本事業は、自治体等が地域課題の解決に向けてローカル5G等の無線通信基盤を活用し、産学官民の連携により具体的な実装を進める取組を支援する政府系広域事業である。通信環境の整備そのものに加え、医療、教育、防災、観光、産業現場などでのユースケース設計、効果検証、運用体制づくりが重視される。申請主体は自治体を中心とし、ミラツクはM+E型の連携事業者として、課題整理、合意形成、データ設計、実装伴走を担う。

## 3. 申請テーマ案（A推奨/B代替・各150字）
A推奨: 「地域知識DBとローカル5Gを活用した自治体横断型・通信DX実装基盤」。学術知識30,288概念と地域現場データを接続し、防災、福祉、観光、産業現場で高精細映像・センサー・遠隔支援を用いた実証を設計する。自治体と企業の協働により、地域課題の可視化から運用モデル化までを一体で進める。

B代替: 「ローカル5Gによる地域産業・公共サービスの遠隔協働モデル構築」。中山間地、公共施設、観光拠点等を対象に、現場映像、設備データ、住民ニーズを統合し、専門人材が遠隔から支援できる仕組みを実証する。ミラツクは知識DBとコミュニティを用いて、利用者視点の要件定義と社会実装を支援する。

## 4. 事業/研究計画書本文（背景目的300-400字、方法300-400字、成果200-300字）
地域では、人口減少、専門人材不足、公共サービスの維持、災害対応、観光地の混雑管理など、現場判断を必要とする課題が同時に進行している。一方、これらの課題は個別部局や単一技術の導入だけでは解決しにくく、現場情報をリアルタイムに取得し、関係者が共有し、実装可能な業務へ落とし込む設計が不可欠である。本申請では、自治体を主体に、ミラツクが保有する26のSQLite知識データベース、学術知識30,288概念、企業協働20社・参加者350名の共創基盤を活用し、ローカル5Gを地域課題解決の通信DX基盤として実装することを目的とする。

方法として、第一に自治体、地域事業者、通信事業者、大学等による実装協議会を組成し、対象地域の課題を業務プロセス、データ、利用者体験の観点から整理する。第二に、ミラツクの学際DBを用いて、国内外の先行事例、関連技術、社会受容性、運用リスクを分析し、ローカル5Gで優先的に解くべきユースケースを選定する。第三に、高精細映像、IoTセンサー、遠隔操作、AI解析等を組み合わせた小規模実証を行い、通信品質、業務削減効果、住民・職員の利用可能性を検証する。第四に、実証結果を踏まえ、自治体が次年度以降に調達・運用できる仕様書、費用対効果、連携体制を整備する。

成果として、対象地域におけるローカル5G活用ユースケースを3件以上設計し、少なくとも1件は実環境での検証まで到達させる。通信基盤の導入に留まらず、自治体業務、地域事業者の運用、住民サービスの改善に結びつく実装モデルを作成する。さらに、知識DBと現場データを統合した評価レポートを整備し、他自治体へ展開可能な地域実装支援パッケージとして公開・提案する。

## 5. スケジュールと予算（年次別）
1年目は、自治体との協定締結、実装協議会の設置、対象課題の選定、既存通信環境の調査、ユースケース設計を行う。予算は人件費、調査設計費、協議会運営費、技術要件整理費を中心に配分する。2年目は、ローカル5G環境の実証、機器・センサー・映像システムの導入、データ収集、効果検証を行う。予算は通信・機器費、実証運営費、データ解析費、外部専門家費に充てる。3年目は、運用モデルの確立、仕様書作成、他地域展開、成果報告会を実施する。予算は成果物作成、自治体展開、広報、継続事業化設計に配分する。
