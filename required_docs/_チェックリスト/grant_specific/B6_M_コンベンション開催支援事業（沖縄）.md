# B6_M_コンベンション開催支援事業（沖縄） 必要書類チェックリスト

## 6. 提出物チェックリスト
- 申請書本体、法人概要、定款、役員名簿、直近決算書、事業報告書を整備する。
- 沖縄県内開催を示す会場見積、開催日程案、参加者規模、海外招聘者リストを添付する。
- フォーラム企画書、プログラム案、広報計画、運営体制図、収支予算書を作成する。
- 沖縄開催の必然性、地域経済波及、宿泊・交通利用、県内連携先を明記する。
- 開催後の報告書、写真・動画記録、参加者数、招聘実績、成果発信計画を提出可能な形にする。
