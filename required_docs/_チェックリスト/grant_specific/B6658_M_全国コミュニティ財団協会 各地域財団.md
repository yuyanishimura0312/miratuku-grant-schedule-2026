+**締切**: 通年 | **上限**: 地域別100-500万
+
+## 1. 申請者情報（300字・数値組込）
+特定非営利活動法人ミラツクは、2010年設立、第14期を迎える非営利法人であり、京都府および東京都を拠点に、地域・企業・研究機関・行政をつなぐ協働型の事業創出に取り組んでいる。専従スタッフ25名を中心に、2024年度営業収益87.4百万円の事業基盤を有し、企業協働20社、コミュニティ参加者350名との継続的な実践を重ねてきた。学術知識30,288概念を含む26のSQLite知識データベースと、連載「事業のかたち」「暮らしのかたち」「変化のかたち」各100話、計300話の知見蓄積を活用し、地域課題を構造化し、実装可能な事業へ転換することを強みとする。
+
+## 2. 助成金概要・応募要件（200字・地の文）
+全国コミュニティ財団協会に加盟する各地域財団の助成は、地域のNPOや市民活動団体等が、地域課題の解決や住民主体の活動基盤づくりを進めるための資金である。募集時期、上限額、対象地域、対象経費は財団ごとに異なるが、通年で相談可能な制度も多く、概ね100万から500万円程度の範囲で、地域内の協働、実装性、継続性、成果の可視化が重視される。申請にあたっては、実施地域の財団方針に合わせて、地域団体との連携体制を明確化する必要がある。
+
+## 3. 申請テーマ案（A推奨/B代替・各150字）
+A推奨「地域課題の可視化と協働事業化支援プログラム」。地域財団、地元NPO、企業、自治体、研究者が参加する対話とデータ分析を組み合わせ、地域課題を整理し、助成後も継続する小規模実証事業を複数立ち上げる。
+
+B代替「地域NPOの事業評価・発信基盤づくり」。地域団体の活動実績、受益者変化、協働関係を整理し、評価指標と発信素材を整備する。財団が支援先の成果を説明しやすくする共通フォーマットを作成する。
+
+## 4. 事業/研究計画書本文（背景目的300-400字、方法300-400字、成果200-300字）
+地域には、孤立、担い手不足、子育て、教育、福祉、移動、文化継承など、分野をまたぐ課題が複合的に存在している。一方で、地域NPOや住民団体の実践知は豊富であるにもかかわらず、課題の構造、関係者の役割、成果の説明が十分に整理されず、継続的な資金や協働につながりにくい状況がある。本事業は、ミラツクが培ってきたセクター協働、データ活用、実装支援、大規模解析の知見を地域財団の現場に接続し、地域課題を「語れる活動」から「支え合える事業」へ転換することを目的とする。地域財団の資金仲介機能を高めるとともに、地域内のNPO、企業、行政、住民が共通の根拠をもって協働できる基盤をつくる。
+
+実施方法は、第一に、地域財団および候補団体へのヒアリングを行い、地域課題、既存活動、協働資源、未充足ニーズを整理する。第二に、ミラツクが保有する26のSQLite知識データベース、学術知識30,288概念、300話の実践知アーカイブを参照し、地域課題を類型化し、類似事例や関連概念を抽出する。第三に、地元NPO、企業、行政、研究者を交えたワークショップを開催し、課題構造図、協働仮説、実証テーマを作成する。第四に、選定した1から3件の小規模実証について、事業設計、評価指標、記録様式、発信文案を伴走支援する。地域財団には、助成先選定や成果説明に活用できる資料一式を提供し、次年度以降の助成設計にも反映できる形で知見を残す。
+
+期待される成果は、地域課題の構造化資料、協働先候補リスト、実証事業計画、成果指標、広報用ストーリーの整備である。これにより、地域NPOは活動価値を説明しやすくなり、地域財団は寄付者や支援者に対して助成の意義を具体的に示せる。さらに、企業協働20社、参加者350名規模のミラツクのネットワークを活用することで、地域外の知見や資源も取り込み、単年度助成にとどまらない継続的な協働モデルを形成する。
+
+## 5. スケジュールと予算（年次別）
+1年目は、1-2か月目に地域財団との協議、対象地域・団体の選定、既存資料の収集を行う。3-4か月目にヒアリングとデータ整理を実施し、5-6か月目にワークショップを開催する。7-10か月目に小規模実証の設計・伴走を行い、11-12か月目に成果資料、評価指標、次年度提案をまとめる。予算は地域別上限に応じ、100万円規模では調査・設計・報告に重点化し、300万円規模ではワークショップと伴走支援を加え、500万円規模では複数団体の実証費、成果発信、外部専門家謝金まで含める。主な費目は人件費、旅費交通費、会場費、謝金、データ整理費、広報制作費、報告書作成費とする。
