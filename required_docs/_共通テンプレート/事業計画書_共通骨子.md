# 事業計画書 共通骨子（助成金申請書「事業計画」セクション標準フォーマット）

助成金申請書における「事業計画」セクションの共通骨子を、ミラツク／esse-senseの実態に即した形でまとめたものです。背景・目的・方法・スケジュール・期待成果・予算・体制の七要素を標準構成とし、各要素を地の文中心で1,500〜2,000字に編集することを想定しています。財団・公的機関ごとの様式差異は`<<<差し替え>>>`タグの箇所で吸収してください。数値は別添「数値ファクトブック.md」と同期させ、強みは「M_E_強み一覧.md」を参照しながら申請テーマに合わせて選択します。

## 冒頭の法人概要（標準引用）

申請主体である特定非営利活動法人ミラツクは、2010年に西村勇也（代表理事）が設立し、第15期（2025年度）を迎えるNPO法人です。京都府および東京都を拠点とし、専従職員25名体制、2024年度営業収益87,400,000円で運営しています。共通の願いから新しい現実を生み出すことをミッションに、企業協働事業（年間20社）、コミュニティ運営（年間350名）、26のSQLite知識データベース構築（学術概念30,288件・関係87,059件）、連載「事業のかたち／暮らしのかたち／変化のかたち」各100話の刊行を主要事業としています。事業会社の株式会社エッセンス（esse-sense, Inc.、2020年12月設立）と研究知識基盤を共有しながら、学術と社会の橋渡しを一貫して担ってきました。
