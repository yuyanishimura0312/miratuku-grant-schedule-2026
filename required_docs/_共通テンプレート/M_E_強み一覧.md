# ミラツク／esse-sense 強み一覧（申請書テーマ4・5用）

申請書の「申請テーマ案」「事業計画」で訴求する7つの差別化点。

## ①学際的データベース基盤
26のSQLite知識DB（学術知識30,288概念ほか）。経営学・イノベーション理論・文学・妖怪・神話narratives・産学連携・規制動態・私立学校・時代別人材・歴史偉人など、人文/社会/自然科学を横断する一次情報資産。**他NPO・他研究機関では模倣困難**。

**助成金で訴求できるテーマ:**
- 文化庁・文部科学省: 文化資源デジタル化、学術基盤整備
- JST RISTEX: SDGs達成、ケア社会システム
- JSPS: データインフラ強化、課題設定先導

## ②セクター横断的協働ネットワーク
年間20社の企業協働、350名のコミュニティ参加者、500名超の研究者ネットワーク。NPO・株式会社・大学・国/自治体・メディアを跨ぐ既存の信頼関係。

**訴求できるテーマ:**
- 経産省: 地域経済循環創造（ローカル10000）
- 内閣府: 新しい地方経済・生活環境創生交付金
- トヨタ財団: 国内助成プログラム

## ③知の社会実装力
連載300話（事業のかたち/暮らしのかたち/変化のかたち）+ 教科書複数冊+ 記事メディア（esse-sense 188本）の発信実績。学術知を一般読者・実務家・政策担当者に翻訳する「Translational Editor」としての訓練済み体制。

**訴求できるテーマ:**
- DNP文化振興財団: グラフィック文化研究
- 京都地域創造基金
- 民間メセナ財団各種

## ④定量データ活用
記事メタデータ、SQLite DB横断検索インデックス、PESTLE/CLA/フォーサイト分析パイプライン。年次/領域別/地域別の定量分析が即座に可能。

**訴求できるテーマ:**
- 総務省: 地域社会DX推進
- 経産省: AI活用関連
- 内閣府: SIP（戦略的イノベーション）

## ⑤実装プロジェクト経験
60超のWebアプリ・自動化ツールの本番運用実績。研究/CRM/業務自動化を統合する開発体制（Claude Code + Codex + 内製スクリプト）。

**訴求できるテーマ:**
- NEDO: スタートアップ支援
- IT導入補助金
- 中小企業省力化投資補助金

## ⑥大規模解析能力
Codex並列処理 + Claude Opus 4.7（1M context）による大規模文書解析、162Kレコードの加速度検証DB、92.9%付与率のGTAコード分類など、**学術データセット級の処理能力**。

**訴求できるテーマ:**
- JST CREST: 戦略的創造研究
- ムーンショット型研究開発事業
- K Program（経済安全保障重要技術）

## ⑦海外研究者招聘ネットワーク
関西エネルギー財団等を通じた海外研究者招聘経験、esse-sense Future Forumでの国際イベント運営、JETROアジアDX等への対応実績。

**訴求できるテーマ:**
- JST ASPIRE: 国際共同研究
- JETRO GSAP: グローバルスタートアップ
- 外務省: 外交安全保障調査研究
